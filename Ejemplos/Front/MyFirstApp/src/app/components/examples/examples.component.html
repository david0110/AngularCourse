Interpolation:

Use it to show any typescript variable in the html. Also used to evaluate any expression and add the result to the html 
{{MyVariable}} or {{2 + 2}}

This is interpolation: {{myVar}}

This is also interpolation: {{2 + 2}}

property binding:

Use it to assign any typescript variable to a property of any tag in the html. 
[value]="MyVarName"
Example: Show the value of the variable `name` in the value of a input: 
<input [value]="name">

Event binding:

Use it to attach an event to a typescript function 
(event)="Function()"
Example: When the user click the button call the function IncreaseCounter 
<input (click)="IncreaseCounter()">
The counter is: {{counter}}

Two way data binding:

Use it when you want a variable not only to be shown but also to be updated. Similar to property binding, but allowing the control to update the value of the variable. If you are showing (interpolating) the variable in another place, that place will be changed when the binding changes the var. 
[(ngModel)]="MyVar"
Example: Show the current value of the var `LastName` but also change the variable when the input is changed. 
<input [(ngModel)]="lastName">

Lastname: {{lastName}}

Directives:

ngFor:

It is used to repeat the same html x amount of times. The loop is controlled with a list variable. 
*ngFor="let variable in listaVariable"
use i to capture the element index: 
let i = index
Example: Iterate the variable studentsList and show it in a table with the index 
<tr *ngFor="let student of studentsList; let i = index">
Index Name Age
{{i}} {{student.name}} {{student.age}}

ngIf:

It is used to Hide and show the html based on the evaluation of any expression. 
*ngIf="ExpressionToShow"
Example: Show when the variable `Show` is true 
<p *ngIf="Show == true"></p>

This text will show or hide when the button is clicked !!

ngClass - ngStyle:

It is used to assign a class or style based on an specific expression. 
[ngClass]="expression ? 'TheClassWhenTrue' : 'TheClassWhenFalse'
Example: Show btn success when the variable `Show` is true, show warning otherwise 
<input class="btn" [ngClass]="Show == true ? 'btn-success' : 'btn-warning'"/>
